<h1>ItemsAdder TypeWriter Extension</h1>

This is a small [TypeWriter](https://modrinth.com/plugin/typewriter) extension for adding [ItemsAdder](https://builtbybit.com/resources/itemsadder.10839/) compatibility.
Current features of the extension include:
- **Item variable entry**: adds the option to get a ItemsAdder item through a NameSpacedID.
- **FontImage variable entry**: adds the option to get a ItemsAdder FontImage through the NameSpacedID.

<h2>Installation</h2>
The installation is just as easy as dropping the jar in the typewriter extensions folder, and you are good to go!
Once you started the webinterface you will the new entries listed.
